Sobre

HTML(HyperText Markup Language)

é a linguagem base da web. Ele define a estrutura de uma página, como títulos, parágrafos, listas, imagens, links e formulários. Funciona como o "esqueleto" de um site.

CSS(Cascading Style Sheets)

é a linguagem usada para estilizar páginas HTML. Com o CSS, controlamos cores, fontes, tamanhos, espaçamentos, posicionamento e responsividade dos elementos.
